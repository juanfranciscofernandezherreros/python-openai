# Spring Boot Article Generator

Este repositorio contiene únicamente la base **Spring Boot** del generador de artículos.

## Módulo principal

- `springboot-article-generator/`: starter/library en Java con autoconfiguración y servicios para generación de artículos.

## Requisitos

- Java 17+
- Maven 3.9+

## Ejecutar tests

```bash
cd springboot-article-generator
mvn test
```
